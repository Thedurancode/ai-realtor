### Method 1: Voice Commands (Telegram Bot) üé§

The easiest way! Just talk to the bot naturally:

**Examples:**

---

### Method 2: Direct API (Technical) üíª

**Add a Note:**
```bash
curl -X POST http://localhost:8000/property-notes/ \
  -H "Content-Type: application/json" \
  -H "X-API-Key: nanobot-perm-key-2024" \
  -d '{
    "property_id": 1,
    "content": "Seller motivated, wants cash buyer",
    "source": "manual"
  }'
```

**Note Sources:**
- `voice` - Added via voice command
- `manual` - Manually entered
- `ai` - Auto-generated by AI
- `phone_call` - From phone call summary
- `system` - System-generated

## Best Practices

### ‚úÖ DO:
- **Be specific** - "Seller wants $850k firm" vs "Price info"
- **Include dates** - "Roof replaced 2023" vs "New roof"
- **Add context** - "Buyer pre-approved $900k, looking in Brooklyn"
- **Update regularly** - Keep notes current
- **Use categories** - Seller, Property, Buyer, Dates, Repairs
- **Link to actions** - "Schedule inspection" not just "Inspection"

### ‚ùå DON'T:
- Vague notes like "Follow up" (when? why?)
- Duplicate information
- Outdated notes (delete or update old ones)
- Irrelevant personal opinions
- Sensitive financial info (keep it professional)
