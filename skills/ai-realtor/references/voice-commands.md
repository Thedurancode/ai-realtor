# Voice Commands Reference

Complete list of all 135 voice commands for AI Realtor platform.

## Using Voice Commands

Voice commands are natural language phrases that the AI assistant understands. You don't need to memorize exact syntax - speak naturally and the AI will match your intent to the appropriate command.

**Example variations all work:**
- "Create a property at 123 Main St"
- "Add a property at 123 Main St"
- "I need to add 123 Main St as a property"

## Tips for Best Voice Command Results

1. **Be specific with property references**
   - Good: "Property 5"
   - Bad: "That property" or "The one we talked about"

2. **Include complete details when creating**
   - Good: "Create property at 123 Main St in Miami for $750,000"
   - Bad: "Create a property"

3. **Use natural language**
   - The AI understands variations, so speak naturally
   - You don't need exact syntax

4. **Context matters**
   - The AI remembers conversation context
   - "Enrich it" works after discussing "property 5"

5. **Multi-step goals**
   - "Set up property 5 as a new lead" → Enrichs, skip traces, attaches contracts, generates recap
   - "Close the deal on property 3" → Checks readiness, completes contracts
